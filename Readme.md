# C Program Template

## Usage with VS Code

### Compile + Debug:

Press `F5`.

### Just Compile:

Press `Ctrl + Shift + B` or invoke a command `Tasks: Run Build Task` from command palette.

## (Alternative Way) Usage with Makefile

### Compile:

```
make
```

### Run:

```
./main
```

### Debug:

```
gdb ./main
```

Or, to enable graphic debugging:

```
gdb -tui ./main
```
